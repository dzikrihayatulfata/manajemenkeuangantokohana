# Keuangan Toko Hana
## Fitur
- Dashboard: Menampilkan total pendapatan, kewajiban, dan investasi.
- Pendapatan: Input pendapatan untuk Toko Hana 1 dan 2.
- Pengeluaran: Kelola pengeluaran supplier, termasuk pembayaran.

Catatan: Ini adalah aplikasi offline. Data hilang jika localStorage dihapus.

Dibuat berdasarkan kode asli yang dipisah menjadi file terpisah untuk kemudahan maintenance.
