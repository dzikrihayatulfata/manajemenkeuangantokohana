# Keuangan Toko Hana

Ini adalah aplikasi sederhana untuk mengelola keuangan toko Hana, dibangun dengan HTML, CSS, dan JavaScript. Aplikasi menggunakan localStorage untuk menyimpan data secara lokal di browser.

## Cara Menggunakan
1. Buat folder baru, misalnya `keuangan-toko-hana`.
2. Salin semua file ke dalam folder tersebut:
   - README.md (ini)
   - index.html
   - styles.css
   - script.js
3. Buka `index.html` di browser Anda (seperti Chrome atau Firefox).
4. Aplikasi siap digunakan. Data akan disimpan di localStorage browser Anda.

## Fitur
- Dashboard: Menampilkan total pendapatan, kewajiban, dan investasi.
- Pendapatan: Input pendapatan untuk Toko Hana 1 dan 2.
- Pengeluaran: Kelola pengeluaran supplier, termasuk pembayaran.

Catatan: Ini adalah aplikasi offline. Data hilang jika localStorage dihapus.

Dibuat berdasarkan kode asli yang dipisah menjadi file terpisah untuk kemudahan maintenance.
